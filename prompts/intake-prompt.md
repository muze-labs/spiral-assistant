# Intake Prompt

Use this prompt when starting with a new project.

```text
Use the Spiral Assistant repository as background.

Help me apply Muze's frontend-first, maturity-gated spiral model to this project.

Start with intake. Ask only for information that is necessary to produce a useful first draft. If information is missing, make reasonable assumptions and mark them clearly.

Produce drafts for:

- MUZE_PROJECT_BRIEF.md
- MUZE_ROADMAP.md
- MUZE_RISK_REGISTER.md
- MUZE_MATURITY.md

Focus especially on:

- project direction;
- roadmap pressure;
- current maturity;
- common web-development risks;
- complexity risk;
- abstraction fitness;
- boundaries;
- AI-era risks;
- evidence needed for the first cycle.

Do not propose a large implementation plan yet. First help us define what risk the next cycle should reduce.
```
